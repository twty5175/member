# member
- MySQL 이용하여 데이터 저장 및 테이블 생성
- PHP 이용하여 정보 삽입, 수정, 삭제 수행

# 과정
1. 회원 입력을 받음
2. 입력받은 데이터를 저장
3. 데이터를 출력
4. 데이터를 수정
5. 데이터를 삭제
